# Auténtico Mijeño

Static bilingual website for property management in Mijas Pueblo, Málaga — vacation rental, long-term rental, and sales.

## Local preview

```bash
python3 -m http.server 4173
```

Open [http://localhost:4173](http://localhost:4173).

## Deploy on Vercel

1. Import this GitHub repository in [Vercel](https://vercel.com/new).
2. Leave the framework preset as **Other** (static site).
3. Deploy — no build command needed.

See `EDITING_GUIDE.md` for how your friend can update copy and images.

# Launch Notes

Auténtico Mijeño is ready to publish as a static site. Upload the project root, keeping `index.html`, `styles.css`, `app.js`, `favicon.svg`, `site.webmanifest`, `robots.txt`, and the `assets/` folder together.

Before connecting the final domain, add the production URL to canonical, `og:url`, and sitemap entries if the hosting provider does not generate them automatically.

Recommended hosting checks:
- Serve over HTTPS.
- Confirm `tel:`, `mailto:`, WhatsApp, and map links open correctly on a phone.
- Replace the mailto form with a hosted form endpoint later if direct inbox delivery is preferred.
